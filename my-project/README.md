# Goldcrest 3D — Next.js նախագիծ

**Սա միայն նախագծի կոդն է** (front + back միասին).

- `app/` — էջեր, layout (frontend)
- `components/` — UI կոմպոնենտներ (frontend)
- `lib/` — DB, validation, services (backend)
- `prisma/` — ԲԴ սխեմա (backend)
- `public/` — ստատիկ ֆայլեր

---

## Մեկնարկ

```bash
cd _project
pnpm install
pnpm dev
```

Բացվում է http://localhost:3000
